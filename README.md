# Estrutura do Projeto

Todo o código-fonte do projeto está localizado na pasta **Projeto**. Nela você encontrará todos os módulos, scripts e componentes necessários para manutenção, leitura e evolução do sistema.

Fora da pasta **Projeto**, está disponível o arquivo **compilado**, pronto para execução. Esse arquivo é o resultado do processo de build e pode ser utilizado diretamente para rodar a aplicação sem necessidade de acessar o código-fonte.

## Resumo
- **/Projeto/** → Contém todo o código do projeto.
- **Arquivo compilado (fora da pasta)** → Usado para executar a aplicação.
